# mood

A RESTful API for mood tracking

![app](data/app.jpeg)

## Getting Started

### Prerequisites

- Java 21 (GraalVM)
- Gradle
- Docker

### Installation

Build the application:

```bash
gradle build
```

Run the application:

```bash
gradle quarkusDev
```
